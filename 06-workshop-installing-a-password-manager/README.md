# Workshop: installing a password manager

THEORETICAL RECAP In the previous lesson on basic security and password management, you discovered why weak or reused passwords are one of the leading causes of account compromise for professionals. A password manager is a dedicated application that securely stores, encrypts, and auto-fills your credentials across all your digital tools. Think of it as a secure vault: you only need to remember one strong master password, and the manager handles everything else. The key mental model is simple — one unique, complex password per account, all protected by one master key. Common mistakes beginners make: using the same password across multiple platforms (a single breach exposes everything), storing passwords in a browser without encryption (easily exported by malware), or writing passwords in a plain text file or spreadsheet. A password manager eliminates all of these risks. For example, instead of using 'Marketing2024!' on both Gmail and Google Drive, your manager generates and stores 'xK9#mPqL2$vR' for Gmail and a completely different string for Drive — both recalled automatically when you log in. Professional digital marketers manage dozens of tools: ad platforms, analytics dashboards, CMS, social media accounts, email marketing software, and more. A password manager is therefore not optional — it is a foundational professional habit. --- WORKSHOP BRIEF This workshop puts into immediate practice the concepts covered in the 'Basic security: password management' lesson.
